# keymou



macos的键盘操纵鼠标软件。 



## 快捷键

## shortcuts

| shortcut                                    | function               |
| ------------------------------------------- | ---------------------- |
| **keymou**                                  |                        |
| Ctrl + opt + `[` / `]` / `;` / `'`          | 鼠标左右下上移动5px    |
| Ctrl + opt + shift +  `[` / `]` / `;` / `'` | 鼠标左右下上滚动10px   |
| Ctrl + opt + \                              | 鼠标移动到下个显示设备 |
| Ctrl + opt + 回车                           | 左键点击               |
| Ctrl + opt + shift + 回车                   | 右键点击               |

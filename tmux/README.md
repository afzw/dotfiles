# tmux

以下P代表Prefix，tmux默认prefix键是C-b，我们改为C-a.

| 操作                    | 快捷键                          |
| ----------------------- | ------------------------------- |
| 窗口导航                |                                 |
| P C-h/l                 | 左/右移动窗口                   |
| 窗格分屏/导航/伸缩/交换 |                                 |
| P -/\|                  | 垂直/水平分窗格                 |
| P h/j/k/l               | 左下上右导航窗格                |
| P H/J/K/L               | 左下右上伸/缩2单位长度          |
| P >/<                   | 当前窗格和下个/上个窗格交换位置 |

## 插件

使用[tpm](https://github.com/tmux-plugins/tpm)管理tmux插件。
